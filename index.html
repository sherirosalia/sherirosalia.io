Recent projects:

Below is a sampling of a few recent projects. They were chosen due to the fact that none of the data is confidential, the visualizations are user friendly and all are deployed on the web. Each image links to the site and the code for these and other projects are posted on github. The link is below.

Geomapping

February 2019

Earthquake data from the US government retrieved via API and used javascript related modules such as leaflet and D3 to create an interactive map that visualizes earthquakes that have occured over the last 10 days. Deployed on Github Pages.

Bacterial Biodiversity

April 2019

Interactive website deployed on heroku that uses python and flask to retrieve data from a RMDS database. Additional modalities were combined in this project including sqlite, javascript, html and css.

Citibike Data Analysis

March 2019

NYC government data retrieved by automating a browser with Python and Beautiful Soup to download csv files, and used Tableau for cleaning and visualization.

The world of data.

It seems that I've become somewhat of a data evangelizer because everywhere I look, life and decisions are better with the ability to analyze information. What is especially exciting about data right now is that not only do we have access to it at an unprecedented level, but additionally, programming languages and tools are phenomenal. My favorite projects are illustrated by the ones above which often involve automating browsers, scraping data from multiple sources and using a variety of programming languages to sort through and visualize the information.
